Project title -
HR ANALYTICS DASHBOARD

DESCRIPTION -
A dynamic, interactive data visualation tool build to explore HR data analysation how many people work, comparision of males and females working, count of employee, attration rate, average salary and average age 

tech stake -
the dashboard using follwing tools 
* power bi desktop - main data visualization use for report creation
* power querry - data transformation and cleaning process
* dax - data analysis expression use for calculated measures ,dynamic visuals and conditional logic
* data modeling - relationship established among tables to enable cross filtering and aggregation
* file format - .pbix for development
